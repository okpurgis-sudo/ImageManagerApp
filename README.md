# 画像管理アプリ

画像フォルダを読み込み、作品ごとにタグ・ジャンル・作者・メモ情報を管理できる、Windows向けの画像ビューアー兼タグ付け検索アプリです。

## 概要

このアプリは、画像フォルダ内のサブフォルダを1作品として扱い、各作品に以下の情報を付けて管理できます。

* 作品名
* シリーズ
* 作者
* ジャンル
* タグ
* ナイスポイント
* 最終閲覧ページ
* ブックマークメモ

画像ファイル本体はアプリ内にコピーせず、選択したフォルダ内の画像をそのまま参照します。

## 主な機能

* 画像フォルダ読み込み
* 作品情報の編集
* 作品名、シリーズ、作者、ジャンル、タグ、ナイスポイントによる検索
* OR / AND 検索
* 空白、半角カンマ、全角カンマ、読点による複数キーワード検索
* グリッドビューのヘッダークリックによる昇順 / 降順ソート
* 数字を含む名前の自然順ソート
* 画像ビューアー
* ページ一覧表示
* ブックマークメモ
* ブックマーク一覧
* 最終閲覧ページの保存
* ライト / ダークモード切り替え
* 前回読み込んだ画像フォルダの自動読み込み

## 対応画像形式

以下の画像形式に対応しています。

* jpg
* jpeg
* png
* bmp
* gif


## フォルダ構成の例

このアプリでは、選択したフォルダの直下にある各サブフォルダを1作品として扱います。

例:

```text
画像フォルダ
├─ 作品A
│  ├─ 1.jpg
│  ├─ 2.jpg
│  └─ 3.jpg
│
├─ 作品B
│  ├─ page1.png
│  ├─ page2.png
│  └─ page3.png
│
└─ 作品C
   ├─ 001.webp
   ├─ 002.webp
   └─ 003.webp
```

この場合、アプリ上では以下のように表示されます。

```text
作品A
作品B
作品C
```

## 画像ビューアーの操作

ビューアー画面では、以下のキー操作ができます。

| 操作    | 内容    |
| ----- | ----- |
| D / → | 次のページ |
| A / ← | 前のページ |
| ↓     | 次の作品  |
| ↑     | 前の作品  |
| Q     | 左回転   |
| E     | 右回転   |

回転はアプリ上の表示だけに反映されます。元の画像ファイルは変更されません。

## 検索について

検索欄では、複数キーワードをまとめて入力できます。

例:

```text
作品A 作品B
作品A,作品B
作品A、作品B
作品A，作品B
```

検索欄の中では、複数キーワードは OR 検索として扱われます。

例:

```text
作品名: 犬 猫
```

この場合、作品名に「犬」または「猫」を含む作品が表示されます。

画面上の OR / AND は、検索欄同士の条件を切り替えるためのものです。

例:

```text
作品名: 犬 猫
タグ: カラー モノクロ
検索モード: AND
```

この場合は、以下の意味になります。

```text
作品名に「犬」または「猫」を含む
かつ
タグに「カラー」または「モノクロ」を含む
```

## 保存されるデータ

アプリの設定や編集情報は、以下のフォルダに保存されます。

```text
C:\Users\ユーザー名\AppData\Local\EBookManagerApp
```

主な保存ファイル:

```text
ebooks.db
theme.txt
last_folder.txt
```

## 注意事項

* 画像ファイル本体はアプリ内に保存されません。
* 画像フォルダを移動または削除すると、アプリから画像を開けなくなる場合があります。
* 編集情報やブックマークはローカルPC内に保存されます。
* GitHubには画像ファイル本体やデータベースファイルを含めないでください。
